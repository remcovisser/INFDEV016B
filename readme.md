#Installation
###To install
`npm install`

Make sure you have nodemon installed:

`npm install nodemon -g`

And gulp

`npm install gulp -g`

###To start server
`npm start`

Compile SASS and client-side JavaScript

`npm dev`

## Or to automatically recompile
`nodemon`

###Requirements
* Node v6.9.1
* MongoDB v3.2.10
* Nodemon v1.11.0
* Gulp v3.9.1
